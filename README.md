> An unofficial, fan-made Foundry VTT system for the **SHIFT Roleplaying Game**, for Foundry V14. A personal project, not affiliated with Hit Point Press. Playing requires the official SHIFT rules.

## Install

**Via Manifest URL (recommended)** — in Foundry, open *Game Systems → Install System*, paste the manifest URL below, and click **Install**:

```
https://github.com/leonebeavi/shift-vtt/releases/latest/download/system.json
```

**Manual download** — download `system.zip` from the [latest release](https://github.com/leonebeavi/shift-vtt/releases/latest) and extract it into `Data/systems/shift-vtt/`.
